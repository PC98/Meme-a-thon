# README

This project was my submission to Georgia Tech's Meme-a-thon, a meme inspired Hackathon.

I made a minimalist website that takes an image from the user and replaces all the faces in the image with random emojis.

### Run Locally
1. Download this repository, and cd into it.
2. Create a folder named 'uploads'.
3. Install all dependencies (list provided at bottom).
4. In the terminal, enter:
```bash
php -S 127.0.0.1:8000
```

5. Browse to [local host](http://127.0.0.1:8000/)
6. Follow the instructions on screen!

### Dependecies
 * Pillow==4.1.0
 * numpy==1.12.1
 * opencv2
 * Python 2.7.13
 * PHP 5.4.0
